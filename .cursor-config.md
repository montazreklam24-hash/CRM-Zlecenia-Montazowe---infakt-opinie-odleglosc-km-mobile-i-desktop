# Konfiguracja CRM - Montaż Reklam 24

## Porty i adresy
- **Dev server (Vite)**: `http://192.168.1.50:3000` - tylko do testów dev
- **Produkcja (PHP/Apache)**: `http://192.168.1.50:8080` - GŁÓWNY PORT DO TESTOWANIA
- **Backend API**: `http://192.168.1.50:8080/api`

## WAŻNE - Zawsze przed testowaniem:
1. **Zbuduj aplikację**: `npm run build:fast` (dla szybkich zmian) lub `npm run build` (z backupem)
2. **Testuj na**: `http://192.168.1.50:8080` (NIE na localhost:3000!)
3. **Hard refresh**: `Ctrl+F5` po każdej zmianie

## Struktura projektu
- Frontend: `src/` → build do `dist/`
- Backend: `api/` → kopiowany do `dist/api/`
- Testowanie: ZAWSZE na `192.168.1.50:8080` po buildzie
